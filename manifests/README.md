# [k8s] Manifests

This directory contains the kubernetes manifests used for deploying the applications that
techwerkers uses on various kubernetes clusters

## Clusters

- `vps` - This is a VPS host, hosted by Hetzner. This contains most public facing applications
- `bramble` - This is a pi cluster hosted by techwerker b-n. These are deemed less priority
  applications due to being hosted on a consumer internet connection.
