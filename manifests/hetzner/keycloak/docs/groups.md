# Keycloak Groups

Below is a description of each of the groups, who should have access, etc

## `listmonk_users`

Gives access to https://listmonk.techwerkers.nl

Note: default access is `campaigner`. An admin in Listmonk can upgrade to `Super Admin` if needed.

## `verified_nohirefire`

A list of users which are verified to not have hiring/firing power

Used for access to:

- http://community.techwerkers.nl
